## 📦 1. 更新系统

首先，更新 WSL2 系统中的包管理器：

```
sudo apt update && sudo apt upgrade -y
```

---

## 🌍 2. 获取 Go 最新版本

前往 Go 官方下载页面：https://go.dev/dl/

找到最新版本，例如目前为：

```
go1.25.1.linux-amd64.tar.gz
```

---

## 📥 3. 下载并安装 Go

```
wget https://go.dev/dl/go1.25.1.linux-amd64.tar.gz
sudo rm -rf /usr/local/go   # 删除旧版本（如果有）
sudo tar -C /usr/local -xzf go1.25.1.linux-amd64.tar.gz
```

---

## ⚙️ 4. 配置环境变量

编辑 `~/.bashrc` 或 `~/.zshrc`（取决于你使用的 Shell）：

```
vim ~/.zshrc
```

在文件末尾添加：

```
export PATH=$PATH:/usr/local/go/bin
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin
```

保存后执行：

```
source ~/.zshrc
```

---

## ✅ 5. 验证安装

```
go version
```

如果输出类似：

```
go version go1.25.1 linux/amd64
```

说明 Go 已安装成功。

---

## 🚀 6. 配置 Go 代理（推荐）

为了加速 Go 模块下载，建议配置国内代理：

```
go env -w GOPROXY=https://goproxy.cn,direct
go env -w GOSUMDB=sum.golang.google.cn
```

这样可以显著提升包下载速度。

---

## 🧪 7. 编写 Hello World 程序

1. 创建项目目录：

   ```
   mkdir -p ~/go/src/hello
   cd ~/go/src/hello
   ```

2. 新建 `hello.go` 文件：

   ```
   vim hello.go
   ```

3. 写入代码：

   ```
   package main

   import "fmt"

   func main() {
       fmt.Println("Hello, WSL2!")
   }
   ```

4. 运行程序：

   ```
   go run hello.go
   ```

你将看到输出：

```
Hello, WSL2!
```

---

## 🎉 总结

到这里，你已经在 **WSL2** 上成功安装并运行了 **Go (Golang)**。
