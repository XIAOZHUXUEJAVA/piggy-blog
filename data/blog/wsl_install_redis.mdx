##  🪟一、更新系统

```bash
sudo apt update && sudo apt upgrade -y
```

------

## 🔧 二、安装 Redis

```bash
sudo apt install redis-server -y
```

安装完成后，可以用以下命令确认版本：

```bash
redis-server --version
```

------

## ⚙️ 三、修改配置（推荐）

默认 Redis 只监听本地 (`127.0.0.1`)，如果你只在 WSL 内使用，可以保持默认。
 如果希望 **从 Windows 或其他设备访问 Redis**，需要修改配置。

打开配置文件：

```bash
sudo vim /etc/redis/redis.conf
```

找到这一行（大约在第 70 行左右）：

```bash
bind 127.0.0.1 ::1
```

改成：

```bash
bind 0.0.0.0
```

并确保以下行被注释掉（以允许远程访问）：

```bash
# protected-mode yes
```

改成：

```bash
protected-mode no
```

> ⚠️ 注意：**如果你要开放远程访问，务必设置密码！**
>  否则任何人都能访问你的 Redis 实例。

设置密码（在同一个配置文件中）：

```bash
requirepass yourStrongPassword
```

保存退出后（`Ctrl + O` → 回车 → `Ctrl + X`）。

------

## 🚀 四、启动与开机自启

启动 Redis 服务：

```bash
sudo service redis-server start
```

查看运行状态：

```bash
sudo service redis-server status
```

开机自启（可选）：

```bash
sudo systemctl enable redis-server
```

------

## 🧪 五、测试 Redis 是否正常

进入 Redis CLI：

```bash
redis-cli
```

测试：

```bash
ping
```

如果返回：

```bash
PONG
```

说明 Redis 工作正常 ✅
