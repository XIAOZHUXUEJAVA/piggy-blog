## 概述

本教程将指导您完成 CodeBuddy Code CLI 的安装过程，并使用 2925 邮箱进行账户注册。

## 前置要求

- 操作系统：Windows 10/11、macOS 或 Linux
- Node.js 20+
- 有效的 2925 邮箱账户

## 步骤一：安装 CodeBuddy Code CLI

### 使用 npm 安装

```bash
npm install -g @tencent-ai/codebuddy-code
```

### 验证安装

运行以下命令验证安装是否成功：

```bash
codebuddy code --version
```

如果显示版本号，说明安装成功。

## 步骤二：使用 2925 邮箱注册 CodeBuddy 账户

### 2.1 准备 2925 邮箱

2925 其实就是一个无限邮箱海量邮箱的邮箱服务，确保您有一个有效的 2925 邮箱账户：
注册地址： https://www.2925.com/

- 邮箱格式：`username@2925.com`, `username1@2925.com`, `username2@2925.com`
- 确保邮箱可以正常接收邮件

### 2.2 注册 CodeBuddy 账户

注册链接： https://www.codebuddy.ai/login

由于注册流程相对简单，这里就不配图片了，直接通过文字说明。使用之前准备好的 2925 邮箱即可无限注册账号。

**版本差异说明：**

- CodeBuddy 国内版：目前似乎可以免费使用
- CodeBuddy 国外版：提供约 1000 点额度，大概可使用 500 次 Claude-4-Sonnet。

**账号通用性：** CodeBuddy Code 和 CodeBuddy IDE 使用相同的账号系统，如果您已经注册过 CodeBuddy IDE，可以直接使用该账号登录 CodeBuddy Code。

**注册建议：**

- 地区选择：建议选择日本地区，目前无需手机号验证
- 后续考虑：如果将来需要手机号验证，日本地区的手机号相比新加坡等地区更容易获取
- 版本推荐：如果计划使用 CodeBuddy IDE，建议注册国外版本(直觉告诉我国内版本有可能会有猫腻, 想想腾讯自家的混元大模型吧哈哈哈)

## 步骤三：配置 CodeBuddy Code CLI

### 3.1 登录 CodeBuddy 账户

终端运行以下命令, 就会提示要去登录了

```bash
codebuddy
```

登录就用刚才注册好的账号就行， 简简单单, 说是注册教程，其实就是一些小tips哈哈~
